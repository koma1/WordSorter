Напишите утилиту командной строки, получающую на вход имя
текстового файла. Она должна прочитать этот файл, составить список
всех слов имеющихся в файле и затем распечатать его в порядке убывания
частоты использования. Словом считайте любой набор алфавитно-цифровых
(юникодных) символов, разделенных не алфавитно-цифровыми символами,
без учета регистра. Файл может быть в разных кодировках
(Ansi/UTF8/UTF16). Основной упор - на эффективность. Должны
обрабатываться любые файлы (влезающие в память), производительность
должна иметь примерно линейную зависимость от размера файла. СУБД
использовать не нужно.

Не торопитесь. Продемонстрируйте правильный подход, а не скорость разработки.
